# Vite & Gourmand

Application web développée dans le cadre de l’ECF Développeur Web et Web Mobile.

Cette application permet la consultation de menus, la gestion des commandes et l’administration d’un service traiteur événementiel.

---

# Technologies utilisées

## Front-end

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Back-end

- PHP 8

## Bases de données

- PostgreSQL
- MongoDB

## Outils

- Git
- GitHub
- Figma

---

# Installation locale

## 1. Cloner le projet

```bash
git clone https://github.com/tissam24/vite-gourmand-ecf.git
```

## 2. Installer les dépendances

```bash
composer install
```

## 3. Configurer la base

Importer les fichiers SQL présents dans :

/sql

## 4. Configurer la connexion

Configurer :

config/database.php
config/mongodb.php

## 5. Lancer

```bash
php -S localhost:8080
```
